Number Scrabble
---
Scrabble with math and numbers. This was created so I could learn and explore [React](https://github.com/facebook/react) and [Webpack](https://webpack.github.io/).

 
Setup
---
 
```
npm install
```


Compile
---
 
```
npm run compile
```
 

Usage
---
 
Start the server with this command:
 
```
npm start
```
